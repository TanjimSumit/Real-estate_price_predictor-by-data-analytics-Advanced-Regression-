# 🏠 Real Estate Price Predictor (Advanced Regression)

A machine learning project that predicts real estate prices using data analytics and regression techniques.

## Project Overview

This project demonstrates a complete data science workflow:

- Data preprocessing
- Feature engineering
- Model training
- Model evaluation
- Price prediction

## Technologies

Python, Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn, Joblib

## Project Structure

## System Workflow

Dataset → Data Preprocessing → Feature Engineering → Model Training → Evaluation → Prediction

## Installation

```bash
git clone https://github.com/TanjimSumit/Real-estate_price_predictor-by-data-analytics-Advanced-Regression-.git
cd Real-estate_price_predictor-by-data-analytics-Advanced-Regression-
pip install -r requirements.txt
```

## Run Model Training

```bash
python src/train_model.py
```

## Run Prediction

```bash
python src/predict.py
```

## Testing

```bash
python -m unittest discover tests
```
